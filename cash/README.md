# cash

Tool to convert currencies

## Installation

```
> git clone https://github.com/romaincaillon/3-musketeers.git
> cd 3-musketeers/cash
> npm i
```

## Usage 

  $ cash <amount> <from> <to>
  $ cash <options>
Options
  --set -s 			Set default currencies
Examples
  $ cash 10 usd eur pln
  $ cash --set usd aud
